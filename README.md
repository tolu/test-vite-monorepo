# test-vite-monorepo

Test setup for vite with monorepo (npm 7 workspaces), using react and sass loading with aliases.

Rough folder structure
```bash
/apps
  /frontend     # the app
    index.html  # app entry-point
  /common       # resources
    src/styles
    src/components
```
